# Word-to-PowerPoint Blueprint

> **Last verified:** 2026-06-01 · **Drift risk:** low

!!! warning "Educational example only"
    This recipe is a workflow pattern for education, not a clinical tool. Its outputs are not clinical decision support and must not inform real patient care. Use only synthetic or published, non-identifiable content — never PHI or restricted research data.

This recipe details the exact workflow to transform detailed, text-heavy lecture notes or Word documents into highly structured, engaging PowerPoint presentation cards using Claude.

---

## The Core Challenge
When converting Word documents to presentations, AI models often create **walls of text** on the slides or structure the slides poorly. A good presentation deck has clear hierarchy, active voice, minimal bullet points, and strong speaker notes.

---

## 3-Step Word-to-PPT Workflow

### Step 1: Prep and Scope the Narrative
Before generating any slide text, ask Claude to map out the structure and narrative flow of the notes.
* **Surface to use:** Claude Chat or a Claude Project.
* **Why it helps:** Reconciles the total length of the document and groups it into logical "chapters" or topics.

**Prompt to use:**
```text
I am converting this Word document / lecture note into a PowerPoint deck.
Before drafting the slides, summarize the content into a 5-topic narrative flow.
Ensure that each topic corresponds to a logical milestone in the presentation.

Source notes:
[PASTE DOCUMENT TEXT]
```

### Step 2: Generate Presentation Cards (Prompt)
Use this prompt to generate the structured slide-by-slide outline. This output format is designed to be highly readable and easy to transfer to PowerPoint.

**Slide Generator Prompt:**
```text
Goal:
Transform the provided lecture notes into a slide outline deck.

Rules for slide copy:
- Slide Title: Bold, active voice, maximum 6 words.
- Slide Content: Maximum 3 bullet points, each under 10 words.
- Visual Suggestion: Describe a simple, high-impact diagram, chart, or split layout. Avoid decorative images.
- Speaker Notes: Reconstruct what I would say aloud in a conversational, professional tone.

Source notes:
[PASTE THE PREPPED CHAPTER/SECTION]
```

### Step 3: Implement the PowerPoint Deck
You have three methods to turn this structured outline into a PowerPoint file:

#### Method A: PowerPoint AI Importer (No-Code)
1. Copy the structured text generated by Claude.
2. In **Word for the Web**, go to `File -> Export -> Export to PowerPoint Presentation`.
3. Choose a design theme and export. It will automatically build the slides based on your headings and bullet points.

#### Method B: Custom python-pptx script (Developer)
If you want to automate deck building, you can have Claude write a quick Python script using `python-pptx` to build the slides.

**Prompt for code generation:**
```text
Write a python-pptx script that takes the following slide outline and builds a clean, structured PowerPoint deck.
Use the default 16:9 widescreen layout.
For each slide, add the Title, the Bullet points, and set a clean font hierarchy (e.g., Arial or Helvetica).

Slide outline:
[PASTE CLAUDE OUTPUT]
```

#### Method C: Google Workspace CLI Plugin (Google Workspace Plugin Workflow)
If you have the `google-workspace-cli` plugin installed:
1. Instruct the CLI agent: `"Read my lecture document from C:/path/to/lecture.docx"`.
2. Instruct the agent: `"Create a new Google Slides presentation based on this structure with 1 slide per topic, then export it as a PowerPoint .pptx file"`.
